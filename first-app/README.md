# Next.js Starter Project 🚀

This is a basic Next.js project to explore routing and nested routes.

## 📁 Project Structure

app/
├── page.js                    --> route: "/"
├── about/
│   ├── page.jsx               --> route: "/about"
│   ├── first/
│   │   └── page.jsx           --> route: "/about/first"
│   └── second/
│       └── page.jsx           --> route: "/about/second"
└── contact/
    └── page.jsx               --> route: "/contact"
